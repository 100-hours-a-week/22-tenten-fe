# 📌 주요 기능
### 1. 소셜 기능
- 게시글/댓글/대댓글 작성
- 게시글/댓글/대댓글 좋아요
- 사용자 팔로우/언팔로우

### 2. 소셜봇 "로로"
- 소셜봇 "로로"와의 채팅
- "로로"의 게시글 작성
- "로로"의 게시글에 작성된 댓글에 대댓글 작성

### 3. 알림 기능
- 팔로우/좋아요 알림
- 댓글/대댓글 알림

### 4. 개인 정보 제공 기능
- 마이페이지
- 프로필 공유 QR
- 프로필 이미지/비밀번호 편집
- 회원 탈퇴
